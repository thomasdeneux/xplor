In the xdata module¶

create_dimension_description¶

xdata.create_dimension_description(label, column)[source]¶

the function creates an instance of DimensionDescription.

create_dimension_description gives an instance of the class DimensionDescription from a label and an column of values of type pandas.core.series.Series.

If column is None, the DimensionDescription instance will be of dimension_type ‘mixed’ by default.

When using this function, no unit is specified, so dimension_description.unit will be None.

Parameters

  • label:
    label for the DimensionDescription instance (type str)
  • column:
    values to determine the dimension_type of the DimensionDescription (type pandas.core.series.Series, shape (n,1))

returns DimensionDescription instance

check_bank_unit¶

xdata.check_bank_unit(unit)[source]¶

The functions checks if this unit is in one of the conversion tables of the bank. If so, it returns the conversion table, else, it returns None

Parameters

  • unit: type str, name of the unit

returns a conversion table for the given unit if it exists in the bank
