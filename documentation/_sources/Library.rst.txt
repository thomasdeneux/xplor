Annexes : Libraries and other tools
***********************************

numpy
+++++

http://www.numpy.org/

NumPy is used as an efficient multi-dimensional container of generic data.
In this project it is used to store the data and easily apply filters on it.

pandas
++++++

http://pandas.pydata.org/

This project uses pandas DataFrame and Series for CategoricalHeader to store
the information corresponding to a line of data in an intuitive way and easily
apply the filters.

Vispy and PyQt
++++++++++++++

http://vispy.org/
https://wiki.python.org/moin/PyQt
https://www.qt.io/

This project uses Vispy and PyQt for the user interface and interactive design.
