Annexes : Libraries and other tools¶

numpy¶

http://www.numpy.org/

NumPy is used as an efficient multi-dimensional container of generic data. In this project it is used to store the data and easily apply filters on it.

pandas¶

http://pandas.pydata.org/

This project uses pandas DataFrame and Series for CategoricalHeader to store the information corresponding to a line of data in an intuitive way and easily apply the filters.

Vispy and PyQt¶

http://vispy.org/

https://wiki.python.org/moin/PyQt

https://www.qt.io/

This project uses Vispy and PyQt for the user interface and interactive design.

Sphinx¶

http://www.sphinx-doc.org/

This project uses Sphinx for the documentation (including autodoc).

Travis CI¶

https://travis-ci.org/

This project uses Travis CI to continuously test the code and deploy only working versions.

WebSequenceDiagrams¶

https://www.websequencediagrams.com/

The sequence diagrams of this documentation were produced using websequencediagrams.

UMLetino¶

http://www.umlet.com/

The UML diagrams of this documentation were produced with UMLetino.
